# Docker-TIL

### 엔트리포인트 (Entrypoint)

- 도커 컨테이너가 실행할 때 고정적으로 실행되는 스크립트 혹은 명령어
- 엔트리포인트가 커맨드 앞에 붙는 방식임 (아래 커맨드에서 echo 엔트리포인트가 커맨드 hello world 앞에 붙음) ; docker run --entrypoint echo ubuntu:focal hello world
